  基于 YOLOv8 的形状检测 + 数字识别 + 距离测量系统
=
【项目简介】
  本项目使用 YOLOv8 目标检测框架，结合摄像头实时画面，实现以下功能：
  1. 检测画面中的 A4 纸板，进行透视校正展开
  2. 识别 A4 纸板内的几何图形（圆形、正方形、三角形），并测量其尺寸（cm）
  3. 识别 A4 纸板内方块上的数字（0~9），根据目标数字定位对应方块，输出尺寸
  4. 通过相机标定（投影变换拟合）推算摄像头到 A4 纸板的实际距离 D（cm）

【使用说明】

▶ 1. 简单推理测试（codes/test.py）
  修改脚本中的模型路径和图片路径后直接运行：
    python codes/test.py
  结果图片保存在 codes/runs/detect/ 目录下。

▶ 2. 形状测量主程序（image.py）

  【首次使用须先标定相机】
    python image.py --model weights/final.pt --cam 0 --res 2k --calib
    - 按照提示，在 120cm、160cm、200cm 处各采样一组数据
    - 按空格键（SPACE）采样，标定结果自动保存到 calib_D.json

  【正常测量模式】
    python image.py --model weights/final.pt --cam 0 --res 2k
    - 将 A4 纸板放入摄像头画面
    - 按 S 键触发一次测量
    - 控制台输出：图形类型、D（距离cm）、x（图形尺寸cm）
    - 按 ESC 退出

  参数说明：
    --model     形状检测模型路径（必填）
    --cam       摄像头编号，默认 0
    --res       分辨率档位：1080p / 2k / 4k，默认 2k
    --imgsz_a4  A4 检测推理分辨率，默认 1536
    --conf      检测置信度阈值，默认 0.15
    --calib     加此参数表示执行标定流程

▶ 3. 数字识别主程序（digit.py）
    python digit.py --shape_model weights/final.pt --digit_model weights/digit2.pt --cam 0 --res 2k
    - 按数字键 0~9 设置目标数字
    - 按 S 键触发测量，程序自动找到对应数字方块，输出尺寸及距离
    - 按 ESC 退出

  参数说明：
    --shape_model    形状检测模型路径（必填）
    --digit_model    数字识别模型路径（必填）
    --cam            摄像头编号，默认 0
    --res            分辨率：1080p / 2k / 4k，默认 2k
    --imgsz_a4       A4 检测推理分辨率，默认 1792
    --conf           检测置信度阈值，默认 0.20
    --target_digit   预设目标数字（也可在运行时按键设置）

--------------------------------------------------------------
【相机标定说明（calib_D.json）】
  使用投影变换模型 D = (a*y + b) / (c*y + d)，
  其中 y 为 A4 底边中点的像素纵坐标，D 为相机到 A4 的距离（cm）。

  当前标定参数对应分辨率：2560 x 1440（2K）。
  若更换分辨率或摄像头，必须重新执行标定（--calib）。

--------------------------------------------------------------
【训练数据说明】
  训练图片及 YOLO 格式标注文件（images/ 和 labels/）
  保存在本目录之外的其他文件夹中，未随项目一起存放。
  如需重新训练，请将数据集路径填写到 data.yaml 对应字段后执行：
    yolo train model=yolov8.yaml data=data.yaml epochs=100 imgsz=640
